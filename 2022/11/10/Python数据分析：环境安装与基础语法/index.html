容器数据类型

容器型数据类型，通俗意义上来说就是一个变量保存多个数据，充当载体的角色。在Python的实际应用中，我们不可能一个变量只赋一个值，应用效率低不说代码也显得臃肿，所以容器型数据类型在Python只扮演着重要的角色。它包括：
列表(list)—— [a,b,c, … ] ，元组(tuple)—— (a,b,c, …) ，字典(dict)—— {“a”: b , …} ，集合—— {a,b,c, …} 以及复数(complex)。

是否有序 是否可变类型 元素重复与否
列表 [ ] 有序 可变类型 可重复
元组 ( ) 有序 不可变类型 可重复
字典 {键值对} 无序 可变（键只能增删，不可修改） 键值对不重复
集合 { } 无序 可变（通过删增实现修改） 不重复

因为 有序 ，可以通过索引（下标）访问元素，所以可以 重复
因为 无序 ，只能通过内容访问元素，所以不可重复
