DQL——数据查询语言

Select语句
select语句除了可以查看数据库中的表格和视图的信息外,还可以查看SQL Server的系统信息、复制、创建数据表。其查询功能强大，是SQL语言的灵魂语句，也是SQL中使用频率最高的语句。

四张表的介绍

1.departments 部门表

department_id: 部门编号
department_name: 部门名称
manager_id: 经理编号
location_id: 区域编号

2.employees 员工表

employee_id: 员工编号
first_name: 名
last_name: 姓
email: 邮箱
phone_number: 电话
job_id: 工种编号
salary: 工资
commission_pct: 奖金率
manager_id: 经理编号
department_id: 部门编号
hiredate: 入职日期

3.jobs 职位表

job_id: 工种编号
job_title: 岗位名称
min_salary: 最低工资
max_salary: 最高工资

4.location 区域表

location_id: 区域编号
street_address: 街道地址
postal_code: 邮政编码
city: 城市
state_province: 州/省
country_id: 国家编号

基础查询

语法

查询员工表的姓名 （多个字段，用逗号隔开 ）

SELECT last_name, first_name FROM employees;

查询多个字段或者全部内容（创建查询工具）

SELECT
employees.employee_id, 
employees.first_name, 
employees.last_name, 
employees.email, 
employees.phone_number, 
employees.job_id
FROM
employees;
或
SELECT * FROM employees;

查询常量值

SELECT 100;
SELECT 'jhon';

查询表达式

SELECT 100%98;

查询 函数

SELECT VERSION();

为表或者字段起别名 AS (方便,区分)

SELECT last_name AS 姓 ,first_name AS 名 FROM employees;
#案例 查询	salary ,显示结果为 OUT put
SELECT salary AS 'OUTput'	FROM employees;

去重 去掉重复值, 查询员工表的所有部门编号

SELECT department_id FROM employees;
SELECT DISTINCT department_id FROM employees;

CONCAT(str1,str2),连接两个字段 姓+名 姓名

SELECT CONCAT(last_name,first_name) AS 姓名 FROM employees;
